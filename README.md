DC-checker

디시인사이드 만화 갤러리(또는 원하는 갤러리)에 특정 제목의 게시글이 올라오면 이메일로 알림을 보내주는 파이썬 스크립트입니다.

🌟 주요 기능

지정된 DCInside 갤러리의 최신 게시글 목록을 크롤링합니다.

게시글 제목에 원하는 키워드가 포함되어 있는지 확인합니다.

키워드가 포함된 새 글이 발견되면 지정된 이메일로 알림을 보냅니다.

🚀 시작하는 법

1. 환경 설정

이 프로젝트는 Python 3가 필요합니다.

먼저, 필요한 라이브러리를 설치합니다.

pip install -r requirements.txt


2. 스크립트 실행

main.py 파일을 수정하여 TARGET_GALLERY_URL과 TARGET_KEYWORDS를 원하는 값으로 변경하세요.

로컬에서 테스트로 실행할 경우, send_email_notification 함수가 이메일 설정값이 없어 콘솔에만 출력된다는 메시지를 보여줄 것입니다.

python main.py


⚙️ 자동화 (GitHub Actions)

이 저장소에는 .github/workflows/main.yml 파일이 포함되어 있습니다. 이 파일은 15분마다 이 스크립트를 자동으로 실행하도록 설정되어 있습니다.

이메일 알림을 받으려면 GitHub 저장소에 Secrets를 등록해야 합니다.

1. (중요) 이메일 서비스에서 '앱 비밀번호' 발급받기

보안상의 이유로, 대부분의 이메일 서비스(Gmail, Naver 등)는 스크립트가 실제 로그인 비밀번호로 접근하는 것을 허용하지 않습니다. 대신 **'앱 비밀번호'**라는 1회용 비밀번호를 발급받아 사용해야 합니다.

Gmail 사용자의 경우:

Google 계정 설정으로 이동합니다.

[보안] 탭으로 이동합니다.

'Google에 로그인하는 방법' 섹션에서 [2단계 인증]이 사용 설정되어 있는지 확인합니다. (안 되어 있다면 설정해야 합니다.)

[2단계 인증] 메뉴 안으로 다시 들어가서, 맨 아래에 있는 [앱 비밀번호]를 클릭합니다.

'앱 선택'에서 [메일]을, '기기 선택'에서 [기타 (맞춤 이름)]을 선택하고 'DC-checker'라고 입력합니다.

[생성] 버튼을 누르면 16자리의 노란색 앱 비밀번호가 표시됩니다. 이것을 복사하세요. (이 창을 닫으면 다시 볼 수 없습니다.)

Naver 사용자의 경우:

네이버 내정보 > [보안설정] > [2단계 인증]으로 이동하여 2단계 인증을 설정합니다.

[2단계 인증] 관리 페이지 하단의 [애플리케이션 비밀번호 생성]에서 [새 비밀번호 생성]을 클릭합니다.

애플리케이션 이름을 'DC-checker' 등으로 입력하고 [생성하기]를 누릅니다.

표시되는 비밀번호를 복사합니다.

2. GitHub Secrets 설정하기

이 GitHub 저장소(orpheus393/DC-checker)의 [Settings] 탭으로 이동합니다.

왼쪽 메뉴에서 [Secrets and variables] > **[Actions]**를 선택합니다.

[New repository secret] 버튼을 눌러 다음 5개의 Secret을 하나씩 등록합니다.

SMTP_SERVER:

Gmail: smtp.gmail.com

Naver: smtp.naver.com

SMTP_PORT: 587 (Gmail, Naver 둘 다 587 또는 465를 사용하나, 587(TLS)을 권장합니다.)

SENDER_EMAIL: 알림을 보내는 이메일 주소 (방금 앱 비밀번호를 발급받은 이메일 주소)

SENDER_PASSWORD: 방금 위에서 발급받은 16자리 앱 비밀번호 (실제 로그인 비밀번호가 아닙니다!)

RECEIVER_EMAIL: 알림을 받을 이메일 주소 (보내는 주소와 같아도 됩니다.)
