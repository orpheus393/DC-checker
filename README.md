DC-checker디시인사이드 만화 갤러리(또는 원하는 갤러리)에 특정 제목의 게시글이 올라오면 알림을 보내주는 파이썬 스크립트입니다.🌟 주요 기능지정된 DCInside 갤러리의 최신 게시글 목록을 크롤링합니다.게시글 제목에 원하는 키워드가 포함되어 있는지 확인합니다.(추후 구현) 키워드가 포함된 새 글이 발견되면 Discord 웹훅 등으로 알림을 보냅니다.🚀 시작하는 법1. 환경 설정이 프로젝트는 Python 3가 필요합니다.먼저, 필요한 라이브러리를 설치합니다.pip install -r requirements.txt
2. 스크립트 실행main.py 파일을 수정하여 TARGET_GALLERY_URL과 TARGET_KEYWORDS를 원하는 값으로 변경하세요.python main.py
3. (중요) CSS 선택자 찾기main.py의 CSS_SELECTOR_FOR_POSTS 값은 갤러리 HTML 구조에 따라 달라질 수 있습니다. 만약 스크립트가 제목을 제대로 가져오지 못한다면, 다음 단계를 따라 직접 찾아야 합니다.크롬 브라우저에서 갤러리 목록 페이지를 엽니다.게시글 제목 중 하나를 마우스 오른쪽 버튼으로 클릭하고 **[검사]**를 선택합니다.개발자 도구에서 해당 제목을 감싸고 있는 태그(보통 <a> 태그)를 찾습니다.해당 태그에서 오른쪽 버튼을 클릭하고 [Copy] > **[Copy selector]**를 선택합니다.복사된 값을 main.py의 CSS_SELECTOR_FOR_POSTS 변수에 붙여넣습니다.⚙️ 자동화 (GitHub Actions)이 저장소에는 .github/workflows/main.yml 파일이 포함되어 있습니다. 이 파일은 15분마다 이 스크립트를 자동으로 실행하도록 설정되어 있습니다.알림을 받으려면 GitHub 저장소의 Settings > Secrets and variables > Actions에 DISCORD_WEBHOOK_URL이라는 이름으로 Discord 웹훅 URL을 등록해야 합니다.⚠️ 주의사항웹 크롤링은 대상 웹사이트의 이용 약관(Terms of Service) 및 robots.txt 파일을 준수해야 합니다. 이 스크립트는 학습 목적으로만 사용하고, 과도한 요청으로 서버에 부담을 주지 않도록 주의하세요.
